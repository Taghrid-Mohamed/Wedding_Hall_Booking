# Wedding_Hall_Booking
 1. About The Project
   This project is a simple Java console application for managing wedding hall booking using Object-Oriented Programming.
 2. Features
  • Add Hall
  • Add Customer
  • Make Booking
  • Display all Halls
  • Display all Customers
  • Display all Bookings
 3. Technologies Used
  • Java
  • OOP
  • NetBeans IDE
 4. Team Members
  • Taghrid
  • Esraa
  • Malak
 5. How to Run
   1- Open the project in NetBeans
   2- Run maint.java
   3- Use the console menu to add data and make bookings
